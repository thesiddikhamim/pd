Pandas

Data analysis and manipulation built on top of NumPy. Fast, powerful, flexible, and easy to use open source data analysis and manipulation tool.

Import & Setup

python
>>>import pandas as pd
>>>import numpy as np  # Best friends
>>>pd.__version__
'2.2.0'
Why as pd? Just like NumPy is np, Pandas is universally imported as pd. You should always import both as they work closely together.

Series

python
>>>s = pd.Series([10, 20, 30], index=['a', 'b', 'c'])
>>>s
a 10 b 20 c 30 dtype: int64
>>>s['b']
20

A Series is a one-dimensional array-like object containing a sequence of values and an associated array of data labels, called its index.

  • If no index is specified, a default integer index (0, 1, 2, ...) is created.
  • Operations between Series automatically align the data based on index labels.
  • Think of it as a fixed-size, ordered dict.

DataFrame

python
>>>df = pd.DataFrame({
... 'A': 1.0,
... 'B': pd.Timestamp('20240101'),
... 'C': pd.Series(1, index=list(range(4)), dtype='float32'),
... 'D': np.array([3] * 4, dtype='int32'),
... 'E': pd.Categorical(['test', 'train', 'test', 'train'])
...})
>>>df.dtypes
A float64 B datetime64[ns] C float32 D int32 E category dtype: object

A DataFrame is a 2-dimensional labeled data structure. You can think of it as a dictionary of Series objects.

  • Columns can have different types (heterogeneous).
  • It has both a row index and a column index.
  • It is the most commonly used pandas object for data analysis.

NumPy Interoperability

python
>>>arr = np.array([[1, 2], [3, 4]])
>>>df = pd.DataFrame(arr, columns=['p', 'q'])
>>>df.values  # Access underlying NumPy array
array([[1, 2], [3, 4]])
>>>np.log(df['p'])  # NumPy ufuncs on Series
0 0.000000 1 1.098612 Name: p, dtype: float64

Pandas and NumPy are built to dance together.

  • Use .values or .to_numpy() to get a NumPy array back from Pandas.
  • Pandas handles alignment, NumPy handles heavy mathematical lifting.
  • Most NumPy functions can take Pandas objects as inputs and return Pandas objects.

Viewing Data

python
>>>df.head(2)  # First 2 rows
Name Age Score 0 Anna 28 88.5 1 Bob 34 92.0
>>>df.describe()  # Summary statistics
Age Score count 3.000000 3.000000 mean 30.333333 88.500000 std 3.214550 3.500000 min 28.000000 85.000000 ...
  • head(n) and tail(n) are your first line of defense for checking data integrity.
  • info() is crucial for spotting missing values and checking memory usage.
  • describe() gives a statistical bird's-eye view, but note it only includes numeric columns by default.

Indexing & Slicing

python
>>>df['Name']  # Select a column
0 Anna 1 Bob 2 Charlie Name: Name, dtype: object
>>>df.loc[0]  # Select row by index label
Name Anna Age 28 Score 88.5 Name: 0, dtype: object
>>>df.iloc[0:2, 0:2]  # Slice by position
Name Age 0 Anna 28 1 Bob 34
  • Square Brackets: df['col'] selects a single column. df[['c1', 'c2']] selects multiple.
  • .loc: Label-based. Use df.loc[row_label, col_label]. Slicing with loc is inclusive of the end.
  • .iloc: Position-based. Works exactly like Python lists or NumPy arrays. Slicing with iloc is exclusive of the end.
Prefer .loc or .iloc for explicit selection over chained indexing (like df[row][col]) to avoid the "SettingWithCopy" warning.

Filtering (Boolean Indexing)

python
>>>df[df['Age'] > 30]
Name Age Score 1 Bob 34 92.0
>>>df[(df['Age'] < 30) & (df['Score'] > 85)]
Name Age Score 0 Anna 28 88.5
>>>df[df['Name'].str.startswith('A')]
Name Age Score 0 Anna 28 88.5

Pandas uses boolean vectors to filter data. Key operators:

  • & (AND), | (OR), ~ (NOT)
  • Always wrap multiple conditions in (parentheses) due to operator precedence.
  • The .str and .dt accessors are powerful for text and date filtering respectively.

Missing Data

python
>>>df.isna().sum()  # Count NaNs per column
Name 0 Age 0 Score 0 dtype: int64
>>>df.fillna(value={'Score': 0, 'Age': 25})
Name Age Score 0 Anna 28 88.5 1 Bob 34 92.0 2 Charlie 29 85.0

Handling "Not a Number" (NaN) is a core data science task.

  • isna() returns a boolean mask. Chaining .sum() is the standard way to check for gaps.
  • dropna() removes rows/columns with missing data. Use subset=['col'] to target specific columns.
  • fillna() provides several strategies (scalar, dict, or method='ffill'/'bfill').

Grouping (Split-Apply-Combine)

python
>>>df.groupby('Age')['Score'].mean()
Age 28 88.5 29 85.0 34 92.0 Name: Score, dtype: float64
>>>df.groupby('Age').agg(['min', 'max', 'mean'])
Score min max mean Age 28 88.5 88.5 88.5 29 85.0 85.0 85.0 34 92.0 92.0 92.0

The Split-Apply-Combine pattern:

  • Split: Data is broken into groups based on key(s).
  • Apply: A function (mean, sum, custom) is applied to each group.
  • Combine: Results are merged into a new object.
Use .agg() for multiple statistics or specific functions per column using a dictionary.

Merging & Joining

python
>>>pd.concat([df1, df2], axis=0)
# Stacks DataFrames vertically (row-wise)
>>>pd.merge(left, right, on='ID', how='inner')
# Database-style join on the 'ID' column
  • concat is for simple stacking (vertical or horizontal).
  • merge is for complex, SQL-like joins. "How" can be 'inner', 'outer', 'left', or 'right'.
  • join is a shortcut for merging on indices.

File I/O

python
>>>df = pd.read_csv('data.csv')
>>>df.to_csv('out.csv', index=False)
# Saved successfully without the index column
>>>df = pd.read_excel('data.xlsx', sheet_name='Sheet1')
  • read_csv / to_csv: Most common for plain text data.
  • read_excel / to_excel: Requires openpyxl or xlsxwriter.
  • read_sql: Directly query databases into DataFrames.
Always use index=False when saving to CSV unless you explicitly want the row numbers saved as a column.

Data Transformation

python
>>>df['Score'].apply(lambda x: x + 5)
0 93.5 1 97.0 2 90.0 Name: Score, dtype: float64
>>>df['Name'].map({'Anna': 'A+', 'Bob': 'B'})
0 A+ 1 B 2 NaN Name: Name, dtype: object
  • apply(): Runs a function along an axis of the DataFrame (or on a Series). Extremely flexible.
  • map(): Used for substituting each value in a Series with another value.
  • applymap(): (Now map() in newer versions) Applies a function to every single element in a DataFrame.

Time Series

python
>>>dates = pd.date_range('20240101', periods=6)
>>>ts = pd.Series(np.random.randn(6), index=dates)
>>>ts.resample('2D').sum()
2024-01-01 -1.234 2024-01-03 0.567 2024-01-05 1.890 Freq: 2D, dtype: float64

Pandas has simple, powerful, and efficient functionality for performing resampling operations during frequency conversion (e.g., converting second data into 5-minute data).

  • date_range: Generates a fixed frequency DatetimeIndex.
  • resample: Essential for frequency conversion and time-series aggregation.
  • tz_localize: Handles time zone information.
